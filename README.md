Readme: 
The program works in two ways.


If an employee logs into the program, it will display the employee’s ID number, name, and birthday. If the employee enters an incorrect ID number or password, the program will terminate.


If a payroll staff member logs in, the staff member can view employee information by either entering an employee ID number or viewing the list of all employees.


If the payroll staff enters an employee’s ID number, they can view the employee’s ID number, name, birthday, and hourly rate. The payroll staff can also view the employee’s net salary per cut-off from June to December.
